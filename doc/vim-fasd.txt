INTRODUCTION                                    *vim-fasd*

This plugin makes the directory jumplist maintained by fasd accessible
from vim.

CHANGE WORKING DIRECTORY                        *vim-fasd-cd*

:Fasd DIR

Change to directory returned by

    fasd -d DIR
